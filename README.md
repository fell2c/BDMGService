# BDMGService
Projeto de construção de um Serviço utilizando HORSE, com SQL Server e a biblioteca RestRequest4Delphi. 

Para funcionamento total do projeto é necessário ter o Boss instalado https://github.com/HashLoad/boss, juntamente com
o Horse https://github.com/HashLoad/horse e o RestRequest4Delphi https://github.com/viniciussanchez/RESTRequest4Delphi.

Necessário fazer a vinculação da pasta console_application e RestRequest4Delphi ao Search Path do vcl_application.
O arquivo Estrutura_MSSQL contém a estrutura utilizada para rodar a API.
